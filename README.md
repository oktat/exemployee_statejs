# exemployee_stetejs

## Beszerzés

```cmd
git clone https://github.com/oktat/exemployee_statejs.git
cd exemployee_statejs
```

## Üzembehelyezés

### Szükséges

* Telepített NodeJS, npm parancs

### Függőségek telepítése

```cmd
npm install
```

### Adatbázis példa beállítása

```cmd
npm run db
```

### API indítása

```cmd
npm run api
```

### Fejlesztői HTTP szerver indítása

```cmd
npm start
```
